# Formats

Formats are the rulesets or play structures that can support a game night. Some are broad families, while others are specific variants with their own setup and components.

## Current Formats

- [BANG-like hidden role formats](bang-like/): Kingdoms, BANG EDH, Treachery, and related secret-role multiplayer variants.
- [Big Deck](big-deck/): Shared-library and pile-style formats.
- [Creature Feature](creature-feature/): Pegasus Loft 60-card constructed with creature-density and shared-type deck-building rules.
- [Cube](cube/): Draft-focused game-night environments.
- [Assassin](assassin/): Target-assignment formats where each player is given a secret mark.
- [Archenemy](archenemy/): One-villain-versus-the-table formats using scheme decks.
- [Horde](horde/): Cooperative players versus an automated enemy deck.

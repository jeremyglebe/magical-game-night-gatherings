# Creature Feature

Creature Feature is a local Pegasus Loft constructed format that uses normal Magic gameplay, but adds deck-building pressure around creature density and shared creature types.

## Core Rules

- Decks have a **60-card minimum**.
- At least **half of the deck** must be creature cards, rounded up.
- At least **two thirds of the creature cards** in the deck must share a creature type, rounded up.
- Games are played as normal Magic unless the event organizer announces an additional house rule.
- Use normal constructed deck-building rules unless the event organizer announces otherwise.

## Creature Type Requirement

Choose one creature type as the deck's featured type. A creature card counts toward the shared-type requirement if its Oracle creature types include the featured type.

For example:

- A 60-card deck needs at least 30 creature cards.
- If that deck has 30 creature cards, at least 20 of those creatures must share the featured type.
- If that deck has 31 creature cards, at least 21 of those creatures must share the featured type.

Creatures with multiple creature types can count for any type they have. Changelings count as having every creature type unless the event organizer has separately banned or restricted that interaction.

## Temporary Type Bans

When a creature type wins a Creature Feature tournament, that creature type should be temporarily banned for the next four weeks.

The goal is to keep dominant types from repeating every event, especially creature types with deep tribal support such as Eldrazi or Sliver. After the temporary ban expires, the type becomes legal again unless it wins another event.

## Negative Creature-Type Hate

Cards that negatively target creature types are banned.

This includes cards whose main purpose is to punish, remove, shrink, prevent, or otherwise disadvantage a named creature type. Color hate, mana-value hate, graveyard hate, artifact hate, and other non-type-specific answers are still legal unless they are banned by another rule.

Use this as the practical test: if the card asks a player to choose or name a creature type and then harms that type or gives protection from that type, it should not be played in Creature Feature.

## Event Notes

- Keep a visible list of currently banned creature types and the date each ban expires.
- Record the winning featured type after each event.
- If a winning deck uses multiple creature types heavily, ban the type that satisfied the two-thirds requirement.
- If the event uses sideboards, the deck should still satisfy the creature-count and shared-type requirements after sideboarding.

# Feudalism

Status: concept sketch.

Feudalism is a [BANG-like hidden-role](../formats/bang-like/) game-night set built around the monarch mechanic.

## Concept

The King role starts the game as the monarch. At the start of the King's turn, if the King is not the monarch, the King becomes the monarch.

The point is that the table can fight over who currently holds power, but power keeps returning to the crown until someone manages to break that structure. A Renegade-style role can win by properly dethroning the King.

## Deck Notes

- All decks should feature monarch cards or care about becoming the monarch.
- Each deck needs to function with any role it is assigned.
- The set should reward fighting over the monarch without making the King impossible to pressure.

## Source Family

This set is currently inspired by the BANG/Kingdom/Treachery family of Commander variants. Keep the detailed source references on the [BANG-like formats page](../formats/bang-like/).
